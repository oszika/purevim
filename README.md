# Pure Vim

## Installation

### Clone repository

#### Vim

```sh
$ git clone https://github.com/oszika/purevim.git ~/.vim
```

#### Neovim

```sh
$ git clone https://github.com/oszika/purevim.git ~/.config/nvim
```

### Install vim-plug

```sh
$ ~/.vim/plugInstall.sh
```

### Install plugins

```
$ vim
:PlugInstall
```
